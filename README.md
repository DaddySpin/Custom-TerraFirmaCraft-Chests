# Custom TerraFirmaCraft Chests
Allows you to add/replace chests that respect TerraFirmaCraft restricted inventory rules. This script does not need BetterEnd or EdenRing installed, script will create the chests in the example even if those mods are not installed.

Maade for Minecraft Forge 1.20.1
Requires KubeJs and TerraFirmaCraft
